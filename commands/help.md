description: タスクから bengo-toolkit の機能を探す対話型メニュー
---

bengo-toolkit には 23 コマンドがあるが、弁護士が覚えるべきなのは `/help` ただ 1 つ。
このコマンドは「今日何をしたい？」をタスク別に聞き、該当する機能へ誘導する。

## $ARGUMENTS の扱い

- **引数なし** → 下記のタスクメニューを表示し、数字または自由記述で選んでもらう
- **`--all`** → 23 コマンド全件を機能カテゴリ別に表形式で出力
- **タスク記述（例: 「離婚調停」「過払金」「戸籍」）** → 該当機能へ直接ナビゲート

## タスクメニュー（引数なしの場合）

以下を日本語でそのまま表示する:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  bengo-toolkit — 今日何をしたい？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. 📄 書類を作成する
     裁判所書式・訴状・示談書・陳述書・財産目録... を PDF から自動入力

  2. 👨‍👩‍👧 相続を整理する
     戸籍謄本から相続関係説明図、法定相続分を分数で計算、遺留分侵害額の算出

  3. ✏️  書面を校正する
     準備書面・契約書の誤字脱字、表記揺れ、法律用語を修正履歴付きでチェック

  4. ⚖️  訴訟文書を分析する
     訴状・答弁書・準備書面からタイムライン・当事者・認否を抽出しレポート生成

  5. 🧮 計算する
     養育費・婚姻費用、財産分与、交通事故損害賠償、残業代、引き直し計算

  6. 📖 法令を調べる
     e-Gov 法令 API で 2,078 法令の条文を検索・参照

  7. 📁 案件フォルダと監査を管理する
     フォルダ単位でテンプレート・監査ログ・出力物を分離管理

  ──────────────────────────────────
  ?. 初めて使う・試したい → /quickstart（60 秒、同梱サンプルで出力を見せるだけ）
  ?. 全コマンド一覧 → /help --all
  ?. 環境を確認 → /verify
  ?. プラグイン更新 → /plugin install bengo-toolkit@bengo-toolkit

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

番号を選ぶか、やりたいことを自由に書いてほしい（例: 「過払金を計算したい」）。
```

## 数字を選んだ場合の分岐

**1. 書類を作成する:**
```
どんな書類を作りたい？
  a. 既に XLSX 書式を持っている → /template-create で登録 → /template-fill で入力
  b. 同梱書式を使いたい（31 種）→ /template-install で選択
  c. 現在の案件フォルダで登録済みの書式を確認 → /template-list
  よく使われる書式:
    財産目録 / 遺産分割協議書 / 訴状 / 答弁書 / 離婚協議書 / 示談書 /
    就業規則 / 株主総会議事録 / 契約書レビューチェックリスト / 委任状 ...
```

**2. 相続を整理する:**
```
相続で何をしたい？
  a. 戸籍から家系図を作る → /family-tree
  b. 法定相続分を計算する → /inheritance-calc
  c. 遺留分侵害額を計算する → /iryubun-calc
  d. 遺産分割協議書を作る → /template-install → /template-fill
```

**3. 書面を校正する:**
```
→ /typo-check <DOCX ファイル名> を実行してほしい。
  対応: 送り仮名、法律用語、及び/並びに/又は/若しくは、当事者呼称、
  法令引用形式、2020年民法改正対応など 362+ ルール。修正は Word 修正履歴で記録。
```

**4. 訴訟文書を分析する:**
```
→ /lawsuit-analysis <訴状.pdf> [答弁書.pdf] [準備書面.pdf] ...
  タイムライン・当事者・主張と認否・証拠対応を .agent レポートにまとめる。
  v2.8.0 以降、分析の立場（原告/被告/中立）をユーザーに確認する Step 3.5 あり。
```

**5. 計算する:**
```
何を計算する？
  a. 養育費・婚姻費用 → /child-support-calc（令和元年改定算定方式）
  b. 離婚時の財産分与 → /property-division-calc（民法 768 条）
  c. 交通事故損害賠償 → /traffic-damage-calc（赤い本基準）
  d. 未払残業代 → /overtime-calc（労基法 37 条・時効 3 年判定）
  e. 引き直し計算（過払金）→ /debt-recalc（利息制限法）
  f. 法定相続分 → /inheritance-calc（民法）
  g. 遺留分侵害額 → /iryubun-calc（民法 1042 条）

  すべて決定論的計算（LLM 推論ではなく確定ロジック）。
```

**6. 法令を調べる:**
```
→ /law-search <キーワードまたは条番号>
  例: /law-search 民法709条
      /law-search 会社法 第362条
      /law-search 不法行為
  略称対応: 民訴法、労基法、民訴規、商登法 など
```

**7. 案件フォルダと監査を管理する（v3.0.0〜）:**
```
v3.0.0 で旧 matter ID 概念は廃止された。**フォルダ = 案件**。

  a. 案件を切替 → cd ~/cases/{別案件} してから claude 起動
  b. 現在の案件詳細を確認 → /case-info
  c. 監査ログ設定を変更 → /audit-config
  d. v2.x のデータを手動移行 → README の「v2 からの移行」参照
```

## `/help --all` を選んだ場合

以下のマトリクスをそのまま表示する:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  全 23 コマンド（カテゴリ別）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【案件フォルダ / workspace】
  /case-info        現在の案件フォルダの状態を表示
  /audit-config     監査ログ設定（記録先・HMAC・クラウド同期）

【テンプレート / template】
  /template-install 同梱 31 書式から選択してインストール
  /template-create  独自 XLSX 書式を登録
  /template-list    登録済み書式の一覧
  /template-fill    PDF からデータを抽出して書式に自動入力
  /template-promote 案件書式を user / firm スコープに昇格（--to user|firm）
  /template-demote  user / firm 書式を案件側へコピー（--from user|firm）
  /template-firm-setup 事務所共有テンプレート用フォルダを設定（OS 同期フォルダ）

【ドキュメント処理】
  /family-tree       戸籍から相続関係説明図を生成
  /typo-check        DOCX の誤字脱字・表記揺れ校正
  /lawsuit-analysis  訴状・答弁書から事件分析レポート生成

【決定論計算器（7 種）】
  /inheritance-calc       法定相続分（民法）
  /traffic-damage-calc    交通事故損害賠償（赤い本基準）
  /child-support-calc     養育費・婚姻費用（令和元年方式）
  /debt-recalc            引き直し計算（利息制限法）
  /overtime-calc          未払残業代（労基法 37 条）
  /iryubun-calc           遺留分侵害額（民法 1042 条）
  /property-division-calc 離婚財産分与（民法 768 条）

【検索】
  /law-search        法令条文を検索（e-Gov API 経由、2,078 法令対応）

【メンテナンス】
  /quickstart        60 秒で試す（同梱サンプル）
  /verify            動作確認
  /plugin install bengo-toolkit@bengo-toolkit  更新（Claude Code 標準）
  /help              このメニュー
```

## タスク記述が自由入力された場合

以下の典型パターンに自動マッチする（部分一致・類義語 OK）:

| 発話パターン | 提案する機能 |
|---|---|
| 「離婚」「離婚調停」「財産分与」 | `/property-division-calc` + `/child-support-calc` + 離婚協議書 template |
| 「相続」「遺産分割」「戸籍」 | `/family-tree` + `/inheritance-calc` + 遺産分割協議書 template |
| 「遺留分」 | `/iryubun-calc` |
| 「過払金」「借金」「債務整理」 | `/debt-recalc` |
| 「残業代」「未払賃金」 | `/overtime-calc` |
| 「交通事故」「人身事故」「損害賠償」 | `/traffic-damage-calc` + 示談書 template |
| 「校正」「誤字」「チェック」 | `/typo-check` |
| 「訴状」「答弁書」「準備書面の分析」 | `/lawsuit-analysis` |
| 「条文」「民法」「会社法」 | `/law-search` |
| 「書式」「テンプレート」「書類作成」 | `/template-install` または `/template-fill` |
| 「初めて」「使い方」 | `QUICKSTART.md` へ誘導 |

マッチしない場合は「タスクメニュー」をそのまま表示する。
