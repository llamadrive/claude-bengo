日本の法令（法律・政令・省令等）を e-Gov 法令 API から検索・取得する。
法令名や条文番号を指定して条文を参照できる。

$ARGUMENTS: 法令名や条文番号（例: "民法 709条", "会社法 第362条", "個人情報保護法"）

まず `skills/law-search/SKILL.md` を Read ツールで読み込み、そこに記載された手順に従って実行する。
