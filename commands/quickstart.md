プロジェクトルートの `QUICKSTART.md` を Read ツールで読み込み、その内容をユーザーに
そのまま表示する。`QUICKSTART.md` には以下が書かれている:

- 前提チェック（/verify と /matter-create の完了確認）
- シナリオ 1: 相続（戸籍 → 相続関係説明図 → 法定相続分 → 遺産分割協議書）
- シナリオ 2: 離婚（財産目録 → 財産分与計算 → 養育費計算）
- シナリオ 3: 交通事故（損害賠償計算 → 示談書）
- よくあるつまづき（表形式）
- フィードバック窓口（GitHub Issues, security メール, 週次レビュー）

表示後、ユーザーに「どのシナリオから始めるか？」を聞く。シナリオを選んだら、その
最初のコマンド（/matter-create 等）の実行を促す。

## 実装

```
Read QUICKSTART.md
→ 全文をユーザーに表示
→ 「どのシナリオから始める？（1: 相続 / 2: 離婚 / 3: 交通事故）」を問う
→ 選ばれたシナリオの Step 1 から対話的に進める
```
