# 令和元年改定 養育費・婚姻費用算定方式リファレンス

東京家裁家事部・大阪家裁家事部の令和元年 12 月 23 日研究報告に基づく
標準算定方式。裁判所ホームページで公開されている算定表（表 1〜表 19）は
この方式から導出される。

## 1. 基礎収入割合

年収に対する基礎収入の割合。給与所得者と自営業者（事業所得）で異なる。

### 給与所得者

| 年収範囲（万円） | 割合 |
|---|---|
| 〜 75 | 54% |
| 〜 100 | 50% |
| 〜 125 | 46% |
| 〜 175 | 44% |
| 〜 275 | 43% |
| 〜 525 | 42% |
| 〜 725 | 41% |
| 〜 1325 | 40% |
| 〜 1475 | 39% |
| 〜 2000 | 38% |

**使う年収**: 源泉徴収票の「支払金額」（額面）を使う。社会保険料や税金を
差し引いた手取りではないことに注意。

### 自営業者（事業所得）

|---|---|
| 〜 66 | 61% |
| 〜 82 | 60% |
| 〜 98 | 59% |
| 〜 256 | 58% |

| 〜 563 | 54% |
| 〜 784 | 53% |

| 〜 1046 | 51% |
| 〜 1179 | 50% |
| 〜 1482 | 49% |
| 〜 1567 | 48% |

**使う年収**: 確定申告書「事業所得」（青色申告の場合は特別控除前）を使う。

**自営業の方が給与より割合が高い理由**: 給与所得者の年収には社会保険料・
通勤費等が含まれており、それらを控除した後の「可処分所得」で計算すべき
ため、給与のほうが低い割合になる。

## 2. 生活費指数

| 区分 | 指数 |
|---|---|
| 親（自活に必要な生活費） | 100 |
| 子 0-14 歳 | 62 |
| 子 15-19 歳 | 85 |

20 歳以上は原則として扶養義務の対象外（民法 877 条）。ただし大学在学中等の
例外あり（その場合は当事者合意または調停で個別に定める）。

## 3. 養育費の計算式

```
義務者基礎収入 = 義務者年収 × 基礎収入割合
権利者基礎収入 = 権利者年収 × 基礎収入割合

子の標準的生活費 = 義務者基礎収入 × Σ(子の指数) / (100 + Σ(子の指数))

義務者分担額（年） = 子の標準的生活費 × 義務者基礎収入 / (義務者基礎収入 + 権利者基礎収入)

養育費月額 = 義務者分担額 / 12
```

計算例: 義務者給与 500 万、権利者給与 100 万、子 1 人（10 歳）

- 義務者基礎収入 = 500 万 × 42% = **210 万**
- 権利者基礎収入 = 100 万 × 46% = **46 万**
- 子の生活費 = 210 × 62/(100+62) = 210 × 62/162 = **約 80.4 万/年**
- 義務者分担 = 80.4 × 210/(210+46) = **約 66.0 万/年**
- 月額 = 66.0 / 12 = 約 55,000 **→ 54,000 円/月**（1,000 円単位丸め後）

算定表（養育費・子 1 人 0-14 歳）の 義務者給与 500 万 × 権利者給与 100 万
のセルは 4〜6 万円。上記計算は範囲内。

## 4. 婚姻費用の計算式

```
権利者世帯の生活費 = (義務者+権利者基礎収入) × (100 + Σ子指数) / (200 + Σ子指数)

義務者分担額（年） = 権利者世帯の生活費 - 権利者基礎収入

婚姻費用月額 = 義務者分担額 / 12
```

計算例: 義務者給与 500 万、権利者給与 100 万、子 1 人（10 歳）

- 義務者基礎 210 万、権利者基礎 46 万
- 権利者世帯生活費 = (210+46) × (100+62) / (200+62) = 256 × 162/262 = 約 **158.3 万**
- 義務者分担 = 158.3 - 46 = 112.3 万/年
- 月額 = 112.3 / 12 = 約 93,600 **→ 94,000 円**（1,000 円単位丸め）

算定表（婚姻費用・子 1 人 0-14 歳）の同セルは 8〜10 万円。計算は範囲内。

## 5. 算定表との対応

算定表は本計算式から以下の手順で作られる:

1. 義務者年収・権利者年収の各範囲セルごとに計算
2. 2 万円単位でセル化（例: 月額 94,000 円 → 「8〜10 万円」）

本計算器は 1,000 円単位で返すため、算定表の「8〜10 万円」枠内により
具体的な値（94,000 円）を示す。実際の調停では算定表の範囲内で
協議・決定することが多い。

## 6. 個別加算事由（本計算器では自動計算しない）

- **住宅ローン負担**: 義務者が家を出て住宅ローンを払い続ける場合、
  その額の一部を義務者の「住居費」として加算する流儀あり
- **私立学校費用**: 公立高校の学習費指数で不足する私立分を加算
- **医療費**: 障害児・慢性疾患等の特別な医療費
- **塾・習い事費用**: 権利者との従前合意があれば加算
- **再婚・養子縁組**: 義務者・権利者の再婚により扶養義務が変動

これらが問題となる場合は、本計算結果を「基礎」として、別途加算分を
検討する実務となる。

## 7. 算定表の適用範囲

| 軸 | 範囲 |
|---|---|
| 義務者年収（給与） | 〜 2,000 万円 |
| 義務者年収（自営） | 〜 1,567 万円 |
| 権利者年収 | 〜 1,500 万円程度 |
| 子の人数 | 1 〜 3 人 |

範囲外の場合、本計算器は警告を出すが計算自体は試みる。実務では
「高額所得者事案」として裁判所が個別に判断することが多い。
