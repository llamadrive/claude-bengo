# 法定相続分の計算ルール（民法）

## 相続人の順位

### 配偶者（民法890条）
- **常に相続人になる**（他の相続人と同順位）
- 離婚済みの元配偶者は相続権なし
- 内縁の配偶者は法定相続権なし（特別縁故者として申立ては可能）

### 第1順位: 子（民法887条）
- 被相続人の子（実子・養子の区別なし）
- 胎児は生まれたものとみなす（民法886条）
- 特別養子縁組の場合: 養親との間のみ相続権あり（実親との相続権は消滅）
- 普通養子縁組の場合: 養親・実親の両方に対して相続権あり

### 第2順位: 直系尊属（民法889条1号）
- 子（代襲者含む）がいない場合のみ
- 父母 → 祖父母 → 曽祖父母の順で、親等の近い者が優先
- 父母の一方のみ生存 → その1人が全額

### 第3順位: 兄弟姉妹（民法889条2号）
- 子も直系尊属もいない場合のみ
- 半血兄弟姉妹（父母の一方のみ共通）は全血の1/2（民法900条4号但書）

## 代襲相続（民法887条2項・3項、901条）

### 要件
- 被相続人の子が**相続開始以前に死亡**した場合
- 相続欠格（民法891条）または廃除（民法892条）の場合
- **相続放棄は代襲原因にならない**

### 効果
- 死亡した子の子（孫）が、死亡した子の相続分を承継する
- 孫が複数いる場合は均等分割
- 孫も死亡している場合はひ孫が再代襲（子の場合は無限に再代襲可能）
- **兄弟姉妹の代襲は1代限り**（甥姪まで。甥姪の子は代襲不可）

## 法定相続分の計算式

### パターン1: 配偶者 + 子
```
配偶者: 1/2
子全体: 1/2 → 子の人数で均等分割
各子: 1/2 ÷ 子の数
```

### パターン2: 配偶者 + 直系尊属
```
配偶者: 2/3
直系尊属全体: 1/3 → 人数で均等分割
```

### パターン3: 配偶者 + 兄弟姉妹
```
配偶者: 3/4
兄弟姉妹全体: 1/4 → 以下のルールで分割:
  全血兄弟: 1単位
  半血兄弟: 0.5単位
  各人の分 = 1/4 × (自分の単位数 / 全単位数の合計)
```

### パターン4: 配偶者のみ
```
配偶者: 1/1（全額）
```

### パターン5: 子のみ（配偶者なし）
```
子全体: 1/1 → 均等分割
```

### パターン6: 直系尊属のみ
```
直系尊属全体: 1/1 → 均等分割
```

### パターン7: 兄弟姉妹のみ
```
兄弟姉妹全体: 1/1 → 半血ルール適用で分割
```

## 代襲相続の計算例

### 例: 被相続人に配偶者 + 子3人（うち1人死亡、死亡した子に孫2人）

```
配偶者: 1/2
子全体: 1/2 → 子3人分（死亡した子の枠も含む）で分割
  子A（生存）: 1/2 × 1/3 = 1/6
  子B（生存）: 1/2 × 1/3 = 1/6
  子C（死亡）の枠 1/6 → 孫2人で均等分割:
    孫C-1: 1/6 × 1/2 = 1/12
    孫C-2: 1/6 × 1/2 = 1/12

合計: 1/2 + 1/6 + 1/6 + 1/12 + 1/12 = 1/1 ✓
```

## 相続放棄の効果

- 放棄した者は**最初から相続人でなかったもの**とみなす
- 放棄者の分は**他の同順位の相続人に配分**される（代襲しない）
- 第1順位の全員が放棄 → 第2順位に移行
- 第2順位も全員放棄 → 第3順位に移行
- 全員放棄 → 相続人不存在

### 放棄の計算例: 配偶者 + 子3人（うち1人放棄）

```
放棄した子は最初からいなかったものとして計算:
配偶者: 1/2
子全体: 1/2 → 残り2人で均等分割
  子A: 1/2 × 1/2 = 1/4
  子B: 1/2 × 1/2 = 1/4

合計: 1/2 + 1/4 + 1/4 = 1/1 ✓
```

## 遺留分（民法1042条）

### 遺留分権利者
- 配偶者、子（代襲相続人含む）、直系尊属
- **兄弟姉妹には遺留分なし**

### 遺留分の割合
- 直系尊属のみが相続人 → 被相続人の財産の **1/3**
- その他の場合 → 被相続人の財産の **1/2**

### 各人の遺留分
```
個人の遺留分 = 遺留分全体 × 法定相続分

例: 配偶者 + 子2人の場合
  遺留分全体: 1/2
  配偶者の遺留分: 1/2 × 1/2 = 1/4
  各子の遺留分: 1/2 × 1/4 = 1/8
```
