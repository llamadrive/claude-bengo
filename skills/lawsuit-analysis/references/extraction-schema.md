# 訴訟分析 抽出スキーマ詳細

## フィールド定義

### summary (string)
文書群全体の概要。約200字。主要な論点や出来事を簡潔に記述する。

### keyPoints (string[])
案件の主要事実を3-7項目でリスト化する。

### timeline (array)

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | string | 一意ID（e1, e2, ...） |
| date | string | 日付。YYYY-MM-DD 形式。月のみ判明の場合は YYYY-MM。年のみの場合は YYYY。期間の場合は YYYY-MM-DD/YYYY-MM-DD。不明の場合は "不明" |
| title | string | 事象の簡潔なタイトル（20字以内） |
| description | string | 事象の詳細説明 |
| category | string | "法的手続き" \| "契約関連" \| "事実関係" \| "その他" |
| importance | number | 重要度 1-10（10が最重要） |

**カテゴリの判断基準:**
- 法的手続き: 訴訟提起、答弁書提出、証拠調べ、判決等
- 契約関連: 契約締結、履行、不履行、解除等
- 事実関係: 事故、出来事、行為等の事実
- その他: 上記に該当しないもの

### characters (array)

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | string | 一意ID（p1, p2, ...） |
| name | string | 正式名称（個人名または法人名） |
| role | string | "原告" \| "被告" \| "証人" \| "弁護士" \| "裁判官" \| "関係者" |
| description | string | 文書中のその人物/組織に関する記述 |
| importance | number | 重要度 1-10 |

### relationships (array)

| フィールド | 型 | 説明 |
|-----------|-----|------|
| source | string | 関係元の人物ID |
| target | string | 関係先の人物ID |
| type | string | 関係の種類（雇用、取引、親族、代理、その他） |
| description | string | 関係の具体的説明 |

### arguments (array)

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | string | 一意ID（a1, a2, ...） |
| title | string | 主張の核心を示すタイトル（30字以内） |
| description | string | 主張内容の詳細 |
| party | string | 主張を行う当事者（"原告" or "被告"） |
| supporting_points | string[] | 主張を支持する根拠・証拠 |
| opposing_points | string[] | 相手方の反論 |
| ninhi | string | 認否ステータス: "認める" \| "否認" \| "一部認める" \| "不知" \| "不明" |

### evidence (array)

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | string | 一意ID（ev1, ev2, ...） |
| party | string | 提出当事者（"原告" → 甲号証, "被告" → 乙号証） |
| number | string | 証拠番号（例: "甲第1号証", "乙第3号証の2"） |
| title | string | 標目（証拠の名称） |
| date | string | 作成日 |
| purpose | string | 立証趣旨 |

### 認否（ninhi）の判定基準

| 値 | 意味 | 判定根拠 |
|-----|------|----------|
| 認める | 相手方の主張を認める | 答弁書・準備書面の認否欄で「認める」と記載 |
| 否認 | 相手方の主張を否認する | 「否認する」「争う」と記載 |
| 一部認める | 一部のみ認める | 「○○の限りで認め、その余は否認する」等 |
| 不知 | 相手方の主張について知らない | 「不知」「知らない」と記載 |
| 不明 | 認否が不明 | 明示的な認否がない、または文書が不足 |

認否は原則として被告側の答弁書・準備書面から抽出する。原告の請求原因事実に対する被告の認否を追跡する。
