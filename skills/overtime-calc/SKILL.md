# 未払残業代計算（overtime-calc）

月別の労働時間記録から、労働基準法 37 条に基づく割増賃金の未払額を
決定論的に計算する。時効（3 年・改正前 2 年）内外を自動区別。

## 前提条件

- アクティブな matter が設定されていること
- 月額基本給（または年俸を 12 分割した額）が判明していること
- 月平均所定労働時間（または年間休日＋日所定労働時間）が判明していること
- 月別の労働時間記録（出勤簿・タイムカード等）が入手できていること

## 割増率（労基法 37 条）

| 種別 | 割増率 |
|---|---|
| 法定時間外（月 60h 以下） | 1.25 倍 |
| 法定時間外（月 60h 超） | 1.5 倍 |
| 深夜（22:00-05:00）所定内 | 1.25 倍（所定賃金が既払なら +0.25） |
| 法定時間外＋深夜 | 1.5 倍 |
| 法定時間外＋深夜（60h 超） | 1.75 倍 |
| 法定休日 | 1.35 倍 |
| 法定休日＋深夜 | 1.6 倍 |

## ワークフロー

### Step 0: Matter 解決

### Step 1: 基礎賃金の聴取

| 項目 | 確認内容 |
|---|---|
| 月額基本給 | 円 |
| 月平均所定労働時間 | 直接入力、または 年間休日+日所定労働時間 から計算 |

**月平均所定労働時間の算定**:
```
月平均所定労働時間 = (365 - 年間休日数) × 1 日所定労働時間 ÷ 12
```

例: 年間休日 120 日、1 日 8 時間 → (245 × 8) / 12 = **163.33 時間/月**

### Step 2: 月別労働時間の聴取

月ごとに以下を整理:

| フィールド | 内容 |
|---|---|
| year_month | YYYY-MM 形式 |
| legal_overtime_h | 法定時間外（月 60h 以下の部分） |
| overtime_over_60_h | 法定時間外のうち月 60h 超の部分 |
| night_h | 所定内労働中の深夜（22:00-05:00）時間 |
| overtime_night_h | 法定時間外＋深夜 |
| overtime_night_over_60_h | 60h 超時間外＋深夜 |
| holiday_h | 法定休日労働 |
| holiday_night_h | 法定休日＋深夜 |

各フィールドは 0 以上の数値（小数点可）。該当なしは 0 または省略。

### Step 3: 時効設定の確認

| 項目 | 内容 |
|---|---|
| statute_years | 3（2020/04 以降発生）／ 2（改正前事案） |
| filing_date | 請求基準日（通常は今日） |

2020/04 改正後に発生した賃金から時効 3 年（労基法 115 条）。

### Step 4: 計算実行

```bash
python3 skills/overtime-calc/calc.py calc --pretty --json '<payload>'
```

入力 JSON 例:

```json
{
  "employee": {
    "name": "甲野太郎",
    "monthly_salary": 260000
  },
  "work_hours": {
    "annual_holidays": 120,
    "daily_scheduled_hours": 8
  },
  "monthly_records": [
    {
      "year_month": "2024-04",
      "legal_overtime_h": 30,
      "night_h": 10
    },
    {
      "year_month": "2024-05",
      "legal_overtime_h": 70,
      "overtime_over_60_h": 10
    }
  ],
  "options": {
    "statute_years": 3,
    "filing_date": "2026-04-01",
    "include_delay_interest": true
  }
}
```

### Step 5: 結果の提示

以下を明示:
- 1 時間あたり賃金（基礎賃金）
- 月別の未払賃金内訳
- 時効内 vs 時効超過分
- 請求可能額合計（時効内 + 遅延損害金）

### Step 6: 注意事項

- 本計算は**固定残業代（みなし残業代）の控除を考慮していない**
- 管理監督者（労基法 41 条 2 号）は割増賃金の対象外
- 裁量労働制・事業場外労働のみなし時間は別途調整が必要
- 家族手当・通勤手当・住宅手当等の除外判定は手計算で確認
- 証拠: タイムカード・出勤簿・業務メール送信時刻等による労働時間の立証が必要

## 対応範囲外

- 固定残業代（みなし残業代）の控除
- 年俸制の割増賃金計算
- 管理監督者の例外判定
- 変形労働時間制（1 ヶ月単位・1 年単位・フレックス）
- 裁量労働制のみなし時間
- 家族手当等の詳細な除外判定

これらが問題となる事案は、計算結果を「未調整の粗計算」と位置づけ、個別に調整する。
